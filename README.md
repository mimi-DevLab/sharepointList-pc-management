# PC端末管理リスト（SharePointカスタムビュー）

> SharePointリストをカスタムJSONフォーマットでギャラリービュー・リストビューに整形したテンプレート

---

## 解決した課題

- 標準のSharePointリストは情報が並列で見づらく、端末ごとの情報が一目でわからなかった
- カスタムJSONフォーマットを適用し、**カード形式・リスト形式の2パターン**で視認性を改善

---

## ビューの種類

### ギャラリービュー（カード形式）
端末1台ずつをカードで表示。使用者・端末ID・パスワードなどをまとめて確認できる。

| 表示項目 | フィールド |
|---|---|
| No. | field_1 |
| 使用者名 | field_2 |
| メールアドレス | field_3 |
| 端末ID | field_4 |
| ID | field_5 |
| パスワード | field_6 |
| 備考 | field_7 |

### リストビュー（行形式）
グループヘッダーに「No. ＋ 合計台数」を表示。一覧での確認・管理に向いている。

## 使い方

### 1. SharePointリストのフィールドを準備する

| フィールド名 | 内部名 | 種類 |
|---|---|---|
| No. | field_1 | 数値 or テキスト |
| 端末ID | field_3 | テキスト |
| 使用者名 | field_4 | テキスト |
| FR ID | field_5 | テキスト |
| メールアドレス | field_6 | テキスト |
| パスワード | field_7 | テキスト |
| 備考 | field_10 | 複数行テキスト |

### 2. ギャラリービューに適用する

1. SharePointリストを開く
2. 右上の **「ビューの切り替え」→「ギャラリー」** を選択
3. 右上の ⚙ **「現在のビューの書式設定」** をクリック
4. **「詳細モード」** を開く
5. `gallery-view.json` の内容を貼り付けて **「保存」**

### 3. リストビューに適用する

1. ビューを **「リスト」** に切り替える
2. 同様に **「現在のビューの書式設定」→「詳細モード」**
3. `list-view.json` の内容を貼り付けて **「保存」**

---

## 工夫したポイント

- グループヘッダーに **「合計：〇台」** を自動表示し、台数をひと目で確認できるようにした
- カード枠に `border-radius` を設定し、SharePoint標準より見やすいデザインに
- 外国籍社員も使う環境を想定し、情報をシンプルに整理したレイアウトを採用
